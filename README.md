# Openair Gewinnspiel Starter

## Lokal starten
npm install
npm run dev

Dann: http://localhost:3000

## Supabase
1. Supabase Projekt erstellen
2. SQL Editor öffnen
3. Inhalt von `supabase-table.sql` ausführen
4. `.env.example` kopieren und in `.env.local` umbenennen
5. Supabase URL und Service Role Key einfügen

## Online stellen
Mit Vercel deployen und dort die Environment Variables eintragen.
